# 杨可一 / One · 科幻产品经理个人网站

这是杨可一（One）的在线作品集网站，采用前卫科幻风格设计，展示其软硬件产品经理的4年复合成长经历、核心能力、项目案例及商业成果。

## 特点

- 🌌 柔和深邃的星际背景，静态星点无闪烁
- 📱 全屏滚动板块，每个板块占据大视口比例
- 🧩 创新布局：不对称网格、大号数字、悬浮玻璃卡片
- 🖱️ 点击卡片弹出详细模态浮窗，展示深度信息
- 📍 左侧页码指示器，实时显示当前板块

## 技术实现

- 纯 HTML / CSS / JavaScript，无外部依赖
- 字体：Google Fonts (Inter, Space Mono)
- 响应式设计，适配移动端

## 部署到 GitHub Pages

1. 将本仓库的所有文件（index.html, README.md, .nojekyll）推送到 GitHub 仓库的主分支。
2. 在仓库设置中启用 GitHub Pages，选择主分支作为源。
3. 访问 `https://<你的用户名>.github.io/<仓库名>/` 即可查看。

> 注意：`.nojekyll` 文件用于禁用 Jekyll 处理，确保纯 HTML 站点正常渲染。
